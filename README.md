## 🔧 Technology Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + Custom Design System
- **UI Components**: Shadcn/ui + Custom Components
- **Audio**: Web Audio API + Custom Binaural Beat Generator
- **Build Tool**: Vite
- **Backend**: Supabase (Database, Auth, Storage)
- **Deployment**: Lovable Platform

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd vitaltones

# Install dependencies
npm install

# Start development server
npm run dev
```

### Environment Setup

The app connects to Supabase for backend functionality. No additional environment variables needed for basic functionality.

## 🎯 Usage

1. **Browse Categories** - Explore different tone categories

5. **Filter** - Filter by category using the category buttons
